# 📚 Quizzy - simple quiz platform

A lightweight and fast platform for creating and taking quizzes.
Easily build custom quizzes, add questions with answer options, and instantly test knowledge.


## 📦 Install

### 1 Clone the repository

```
git clone https://github.com/acoji-luo/quiz-webapp.git
cd quiz-webapp
```

### 2 Clone the repository

```
yarn
# or
npm i
```

### 3 Environment variables
Create a .env file in the project root:
```
VITE_API_URL=http://localhost:5000
```

### 4 Run in development mode
```
yarn dev 
# or
npm run dev
```
